# Project Instructions

## Python Environment

- Use the repository-local virtual environment for Python work.
- On Windows/PowerShell, run Python as `.\.venv\Scripts\python.exe`.
- Install packages with `.\.venv\Scripts\python.exe -m pip install -r requirements.txt`.
- Do not use the global Python interpreter for project scripts unless the local `.venv` is missing or broken; if that happens, report the blocker before continuing.
